This directory contains tools and scripts useful for this project.

 Makefile maker (`makemake.sh`
------------------------------
Generate makefile for Unix type systems.

 Pack script (`pack.cmd`)
--------------------------
This is a Windows CMD script for packing the whole repository
into a `rawdev.tgz` file. Temporally and intermediate files and
directories are skipped. This is useful when we want pack and
backup the project manually.

See the script source for usage.
