# Test directory

This directory contains data for testing purposes. The dataset 
is very large, so I decided to put only one here, as the other 
may be downloaded from the internet. So we don't blow up the 
project repository with binary files that we can get elsewhere.

 Where to get more images
--------------------------
As I&nbsp;cannot own all these cameras, I must rely on
other sources. Most of these RAW files can be downloaded
from [imaging-resource](https://www.imaging-resource.com/).
So special thanks, as they made it possible to test files from
a&nbsp;variety of different professional cameras.

 Important notes
-----------------
*Make sure you use full RAWs* (not mRAW or sRAW). These small RAWs
are not compatible, as their structure is totally different (and worse). 
The reason for this is that no one who cares about quality would use them. 

Also make sure that the full RAWs are from one of the supported cameras 
(see the main readme). Every camera has its own color calibration profiles,
therefore we have a&nbsp;limited set of supported cameras.
